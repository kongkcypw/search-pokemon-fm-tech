# Chayapol Wongpuwarak

FM Full Stack Developer Test Submission

## Key features

- User can search pokemon by name with suggestions

- User can see pokemon details, attacks, and evolutions

- User can mark pokemon as favorite and see list of favorite pokemon

## Showcasing

- Next.js routing (using both static and dynamic routing)

- Apollo Client for GraphQL and use cache for pokemon list

- Static optimization for featured pokemon on home page

- React features (useState, useMemo, useEffect, hooks)

- LocalStorage for favorite pokemon

## Open on your browser

https://search-pokemon-fm-tech-silk.vercel.app/

## Run locally

Follow these steps to set up and run the project locally.

### Installation

To install the dependencies, run:

```bash
npm install
```

### Running the Application

To start the development server, run:

```bash
npm run dev
```

## Unit Tests

- `npm run test`: Runs the test suite using Jest, test only pokemon type.

## Lint

- `npm run lint`: Runs the linting process using ESLint.
